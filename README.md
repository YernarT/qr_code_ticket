# QR Ticket

> QR кодға негізделген билет жүйесі

## How to use

- Start frontend server
    - `cd frontend`
    - `npm install` (first open)
    - `npm run dev`

- Start backend server
    - `cd backend`
    - `pip install -r requirements.txt` (first open)
    - `python manage.py migrate` (first open)
    - `python manage.py runserver`
